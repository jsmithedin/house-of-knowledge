# UV Migration Design

**Date:** 2026-06-12  
**Status:** Approved

## Goal

Replace the two pip-based requirements files with a single uv project. The eval dependencies become an optional extras group so a plain `uv sync` installs only the main app, and `uv sync --extra eval` adds the eval tooling on top.

## Project structure

| Before | After |
|--------|-------|
| `requirements.txt` | `pyproject.toml` (authoritative) |
| `eval/requirements-eval.txt` | `[project.optional-dependencies].eval` in `pyproject.toml` |
| _(none)_ | `uv.lock` (committed) |

`pytest` moves to `[dependency-groups].dev` — standard uv pattern, keeps it out of the production Docker image.

## `pyproject.toml` shape

```toml
[project]
name = "house-of-knowledge"
version = "0.1.0"
requires-python = ">=3.11"
dependencies = [
    "streamlit==1.44.1",
    "chromadb==0.6.3",
    "sentence-transformers==3.4.1",
    "boto3==1.37.29",
    "python-frontmatter==1.1.0",
    "pyyaml==6.0.2",
]

[project.optional-dependencies]
eval = [
    "tabulate==0.9.0",
]

[dependency-groups]
dev = [
    "pytest==8.3.5",
]
```

## Dockerfile

- Copy the uv binary from `ghcr.io/astral-sh/uv:latest` (unpinned).
- Replace `pip install -r requirements.txt` with `uv sync --frozen --no-dev`.
- Pre-download BGE-M3 via `uv run python -c ...`.
- Run streamlit via `uv run streamlit run ...` in CMD.

```dockerfile
FROM python:3.11-slim

WORKDIR /app

RUN apt-get update && apt-get install -y --no-install-recommends \
    build-essential \
    && rm -rf /var/lib/apt/lists/*

COPY --from=ghcr.io/astral-sh/uv:latest /uv /usr/local/bin/uv

COPY pyproject.toml uv.lock ./
RUN uv sync --frozen --no-dev

RUN uv run python -c "from sentence_transformers import SentenceTransformer; SentenceTransformer('BAAI/bge-m3')"

COPY app/ app/
COPY scripts/ scripts/

EXPOSE 7860
CMD ["uv", "run", "streamlit", "run", "app/main.py",
     "--server.port=7860",
     "--server.address=0.0.0.0",
     "--server.headless=true"]
```

## eval/README.md updates

**Prerequisites section** — replace:
```bash
pip install -r eval/requirements-eval.txt
```
with:
```bash
uv sync --extra eval
```

**Workflow table** — all stage commands get `uv run` prefixed:

| Stage | Command |
|-------|---------|
| 1 | `uv run python -m eval.runner --run-id YYYY-MM-DD-name` |
| 2 | `uv run python -m eval.score_retrieval --run-id YYYY-MM-DD-name` |
| 3 | `uv run python -m eval.make_blind --run-id YYYY-MM-DD-name` |
| 4 | `uv run streamlit run eval/scoring_app.py -- --run-id YYYY-MM-DD-name` |
| 5 | `uv run python -m eval.unmask --run-id YYYY-MM-DD-name` |
| 6 | `uv run python -m eval.judge --run-id YYYY-MM-DD-name` |
| 7 | `uv run python -m eval.report --run-id YYYY-MM-DD-name` |

The `--limit` smoke test example also gets `uv run`:
```bash
uv run python -m eval.runner --run-id 2026-06-15-smoke --limit 2
```
